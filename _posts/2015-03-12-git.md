###一些git命令

记录一下自己学习的一些git命令

1. 从远程仓库进行更新

    $ git fetch origin
	
	$ git merge origin/master

为什么不用git的pull，参考 [Git 少用 Pull 多用 Fetch 和 Merge][1]

2. 把你的变更推送到一个远程仓库

	$ git push origin master
